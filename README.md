
# Amazon EC2 (Elastic Compute Cloud)

## O que é o Amazon EC2

O Amazon EC2 (Elastic Compute Cloud) é um serviço da Amazon Web Services (AWS) que oferece capacidade de computação na nuvem sob demanda. Ele permite que usuários criem e utilizem servidores virtuais, chamados de instâncias, para executar aplicações, sites e sistemas sem a necessidade de adquirir hardware físico.

---

## Como funciona

O funcionamento do EC2 é simples e flexível:

* O usuário cria uma instância (máquina virtual) na nuvem
* Escolhe o sistema operacional (Linux, Windows, etc.)
* Define os recursos da máquina (CPU, memória e armazenamento)
* Executa aplicações ou hospeda serviços nessa instância
* Paga apenas pelo tempo em que a instância estiver em uso (modelo pay-as-you-go)

---

## Principais características

### Escalabilidade

Permite aumentar ou reduzir os recursos de forma rápida conforme a necessidade da aplicação.

### Flexibilidade

Oferece diversos tipos de instâncias, possibilitando escolher configurações específicas para diferentes cargas de trabalho.

### Controle total

O usuário tem controle completo sobre o sistema operacional, configurações de rede e segurança.

### Alta disponibilidade

As instâncias podem ser distribuídas em diferentes regiões e zonas de disponibilidade da AWS, garantindo maior confiabilidade.

---

## Para que serve

O Amazon EC2 é utilizado em diversas situações, como:

* Hospedagem de sites e aplicações
* Desenvolvimento e testes de software
* Execução de sistemas corporativos
* Processamento de grandes volumes de dados

---

## Conclusão

O Amazon EC2 pode ser entendido como um computador virtual na nuvem, que pode ser utilizado sob demanda. Ele elimina a necessidade de infraestrutura física própria, oferecendo uma solução escalável, flexível e com custo baseado no uso.
